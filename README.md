# Arduino 09012020
初始指令：
void setup() <--> 只執行一次,Arduino的初始設定
vid loop() <-->會重複執行,主程式的位置
第一個指令：
pinMode (pin,INPUT/OUTPUT)
此語法為設定腳位輸出(OUTPUT)或輸入(INPUT)
INPUT的初始狀態 = 1
OUTPUT的初始狀態 = 0
